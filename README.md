# Unity Procedural Map Generator

A procedural map generation system built in Unity using C# and tile-based logic.

## Overview

This project focuses on generating game maps dynamically using algorithmic techniques instead of manually designing levels.

It is designed as a reusable system that can be integrated into different games.

## Features

- Procedural tile-based map generation
- Randomized terrain creation
- Modular and reusable architecture
- Easily extendable for different game types
- Clean separation of generation logic

## Technologies

- Unity
- C#
- Tilemap system

## Project Structure

- Assets/Scripts → core generation logic
- Assets/Scenes → demo scene
- ProjectSettings → Unity configuration

## Usage

Open the project in Unity and run the scene to see the generated map.

## Future Improvements

- Biome-based generation
- Pathfinding integration
- Enemy spawn system
- Optimization for large maps
